# Contributing to Base Ecosystem Research

## How to Contribute

### Adding Projects
1. Verify project is on Base mainnet
2. Include: Name, Type, Status, Why it matters
3. Add airdrop potential rating
4. Include actionable steps

### Updating Information
- Check links are active
- Update TVL numbers quarterly
- Mark projects as live/testnet/deprecated

### Research Standards
- Verify all claims
- Include sources where possible
- Be objective about risks

## Format

Use this template for new projects:

```markdown
### Project Name (URL)
- **Type**: Category
- **Status**: Live/Testnet/Coming Soon
- **TVL**: $X (if applicable)
- **Why it matters**: One sentence
- **Airdrop potential**: HIGH/MEDIUM/LOW/NONE
- **Action**: What to do
```
